API

获取验证码

从12306获取一张测试的验证码

  • Request

    • Method: GET
    • Headers
      • Content-Type: application/json
    • URL: /Mark12306Captcha/api/v1.0/get/captcha

  • Response

    • Headers
      • Content-Type: application/json
    • Body

识别

Full

  • Request

    • Method: POST
    • Headers
      • Content-Type: application/json
    • URL: /Mark12306Captcha/api/v1.0/mark
    • Body
      • {"originCaptcha":"captcha base64 string"}

  • Response

    • Headers
      • Content-Type: application/json
    • Body

  • Result 解释说明

    • ids 图片的编号
      • 0 2 4 6
        1 3 5 7
    • results 一个列表，每个元素是标记的坐标
    • markedCaptcha 绘制好标记结果的验证码Base64字符串

Lite

  • Request

    • Method: POST
    • Headers
      • Content-Type: application/json
    • URL: /Mark12306Captcha/api/v1.0/mark/lite
    • Body
      • {"originCaptcha":"captcha base64 string"}

  • Response

    • Headers
      • Content-Type: application/json
    • Body
    • Result 解释说明参见上面
